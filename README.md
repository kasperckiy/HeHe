# HeHe

Расширение для hh.ru, которое помогает быстрее чистить выдачу вакансий и сохранять полезные тексты для отклика.

Оно добавляет быстрые кнопки скрытия вакансий и компаний, а также даёт отдельную память заметок, куда можно сохранять сопроводительные письма и ответы из форм отклика.

## Для чего это нужно

- чтобы быстро убирать из выдачи неинтересные вакансии
- чтобы не видеть повторно вакансии со статусами `Вы откликнулись`, `Вас пригласили`, `Вам отказали`
- чтобы хранить готовые сопроводительные письма и другие заметки для отклика в одном месте
- чтобы сохранять тексты прямо из формы отклика на hh.ru и потом использовать их повторно

## Что получает пользователь

- работает на страницах поиска вакансий hh.ru
- добавляет вторую кнопку рядом с кнопкой отклика
- позволяет скрывать отдельные вакансии и компании
- умеет по переключателю в popup автоматически скрывать карточки со статусами `Вы откликнулись`, `Вас пригласили`, `Вам отказали`
- добавляет на карточки с этими статусами отдельную кнопку `Скрыть`
- на странице вакансии показывает статусы `Вакансия скрыта` и `Работодатель скрыт`, а также кнопки скрытия
- открывает отдельный экран с сохранёнными заметками через popup расширения
- добавляет в форму сопроводительного письма кнопку выбора сохранённой заметки
- умеет сохранять тексты из форм отклика через inline-переключатель `Сохранить на потом`
- поддерживает сохранение сопроводительного письма и, при включённой настройке, кастомных полей работодателя

## Как установить

### Вариант 1. Из GitHub Release

1. Открой страницу `Releases` в этом репозитории
2. Скачай архив последнего релиза `HeHe-vX.Y.Z.zip`
3. Распакуй архив в отдельную папку
4. Открой `chrome://extensions`
5. Включи `Режим разработчика`
6. Нажми `Load unpacked`
7. Выбери папку, в которую был распакован архив

Важно:
Chrome в режиме разработчика не умеет подключать `.zip` напрямую, поэтому архив нужно сначала распаковать.

### Вариант 2. Из исходников

1. Скачай или клонируй репозиторий
2. Открой `chrome://extensions`
3. Включи `Режим разработчика`
4. Нажми `Load unpacked`
5. Выбери папку проекта
6. Нажми на иконку расширения и открой экран `Заметки отклика`
7. При необходимости настрой автоскрытие статусов, логирование и сохранение кастомных полей формы

## Как пользоваться

### 1. Скрывать вакансии в поиске

- в карточке вакансии нажми `Не интересно`, если не хочешь видеть её снова
- если карточка уже имеет статус отклика, можно нажать `Скрыть`
- при включённом автоскрытии такие карточки будут убираться автоматически

### 2. Скрывать вакансии и компании на странице вакансии

- на самой странице вакансии используй `Скрыть вакансию` или `Скрыть компанию`
- если объект уже скрыт, расширение покажет это отдельным статусом

### 3. Хранить заметки

- открой popup расширения
- нажми `Открыть заметки`
- создай заметку с названием и текстом
- заметки можно редактировать и удалять

### 4. Подставлять заметки в форму отклика

- в поле сопроводительного письма нажми `Выбрать из моих заметок`
- выбери нужную заметку из списка
- если нужно, открой полный экран заметок через `Посмотреть все`

### 5. Сохранять текст из формы отклика

- под нужным `textarea` включи `Сохранить на потом`
- задай название заметки
- нажми `Откликнуться`
- при отправке формы расширение сохранит только те поля, где была отмечена галка

## Безопасность и данные

- расширение не отправляет ваши заметки, токены, cookies, резюме, отклики и другие персональные данные на сторонние серверы
- заметки хранятся локально в `chrome.storage.local` внутри браузера
- запросы из расширения отправляются только на сам `hh.ru`, чтобы скрывать вакансии и получать их текущий статус скрытия
- расширение не содержит собственного удалённого backend, аналитики или облачного хранилища

## Технические детали

- скрытие вакансий и компаний работает через внутренние blacklist API hh.ru
- состояние скрытия на странице вакансии тоже проверяется через hh.ru API
- список вакансий и формы отклика отслеживаются на динамически изменяющейся странице через `MutationObserver`
- popup открывает отдельную страницу заметок через стандартные возможности Chrome extension
- хранение заметок реализовано через `chrome.storage.local`

## Ограничения

- разметка hh.ru может меняться, поэтому часть логики сделана с fallback-селекторами
- если hh.ru сильно изменит разметку формы отклика, селекторы для inline-заметок могут потребовать обновления
