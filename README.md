# HeHe

Расширение для hh.ru для Chromium-браузеров и Firefox, которое помогает быстрее чистить выдачу вакансий и сохранять полезные тексты для отклика.

Оно добавляет быстрые кнопки скрытия вакансий и компаний, а также даёт отдельную память заметок, куда можно сохранять сопроводительные письма и ответы из форм отклика.

## Для чего это нужно

- чтобы быстро убирать из выдачи неинтересные вакансии
- чтобы не видеть повторно вакансии со статусами `Вы откликнулись`, `Вас пригласили`, `Вам отказали`
- чтобы хранить готовые сопроводительные письма и другие заметки для отклика в одном месте
- чтобы сохранять тексты прямо из формы отклика на hh.ru и потом использовать их повторно

## Что получает пользователь

- отдельные архивы для Chromium-браузеров и Firefox
- Chromium-архив подходит для Chrome, Edge и Opera
- Firefox-архив подходит для Firefox
- работает на страницах поиска вакансий hh.ru
- добавляет вторую кнопку рядом с кнопкой отклика
- позволяет скрывать отдельные вакансии и компании
- умеет по переключателю в popup автоматически скрывать карточки со статусами `Вы откликнулись`, `Вас пригласили`, `Вам отказали`
- добавляет на карточки с этими статусами отдельную кнопку `Скрыть`
- на странице вакансии показывает статусы `Вакансия скрыта` и `Работодатель скрыт`, а также кнопки скрытия
- открывает отдельный экран с сохранёнными заметками через popup расширения
- добавляет в форму сопроводительного письма кнопку выбора сохранённой заметки
- умеет сохранять тексты из форм отклика через inline-переключатель `Сохранить на потом`
- поддерживает сохранение сопроводительного письма и, при включённой настройке, кастомных полей работодателя
- умеет по кнопке `Переписать с Gemini` переписывать текст в подходящих `textarea` формы отклика
- позволяет выбрать Gemini API key, модель и режим переписывания в popup расширения

## Поддерживаемые браузеры

- Chrome: поддерживается через пакет `HeHe-chromium-vX.Y.Z.zip`
- Edge: поддерживается через пакет `HeHe-chromium-vX.Y.Z.zip`
- Opera: поддерживается через пакет `HeHe-chromium-vX.Y.Z.zip`
- Firefox: поддерживается через пакет `HeHe-firefox-vX.Y.Z.zip`
- Safari: пока не поддерживается

## Как установить

### Вариант 1. Из GitHub Release

Последний релиз:
<https://github.com/kasperckiy/HeHe/releases/latest>

1. Открой страницу `Releases` в этом репозитории
2. Скачай архив под свой браузер:
   - `HeHe-chromium-vX.Y.Z.zip` для Chrome, Edge и Opera
   - `HeHe-firefox-vX.Y.Z.zip` для Firefox
3. Распакуй архив в отдельную папку
4. Установи расширение по браузеру:
   - Chrome: открой `chrome://extensions`, включи `Режим разработчика`, нажми `Load unpacked` и выбери распакованную папку
   - Edge: открой `edge://extensions`, включи `Режим разработчика`, нажми `Load unpacked` и выбери распакованную папку
   - Opera: открой `opera://extensions`, включи `Режим разработчика`, нажми `Load unpacked` и выбери распакованную папку
   - Firefox: открой `about:debugging#/runtime/this-firefox`, нажми `Загрузить временное дополнение` и выбери `manifest.json` из распакованной папки

Важно:
Браузеры не подключают `.zip` напрямую, поэтому архив нужно сначала распаковать.

Важно для Firefox:
Установка через `about:debugging` является временной. После перезапуска Firefox расширение нужно загрузить заново, пока оно не опубликовано в магазине дополнений.

На каждом теге `v*` GitHub Actions автоматически собирает оба ZIP-архива и прикладывает их к релизу.

### Вариант 2. Из исходников

1. Скачай или клонируй репозиторий
2. Запусти `npm run build:packages`
3. Выбери собранную папку из `dist`:
   - `dist/chromium` для Chrome, Edge и Opera
   - `dist/firefox` для Firefox
4. При необходимости можно собирать таргеты отдельно:
   - `npm run build:chromium`
   - `npm run build:firefox`
5. Установи расширение так же, как в варианте из релиза
6. Для Firefox загружай `dist/firefox/manifest.json` как временное дополнение
7. Нажми на иконку расширения и открой экран `Заметки отклика`
8. При необходимости настрой автоскрытие статусов и сохранение кастомных полей формы
9. Если хочешь использовать Gemini rewrite, добавь API key, выбери модель и режим переписывания в popup расширения

## Как пользоваться

### 1. Скрывать вакансии в поиске

- в карточке вакансии нажми `Не интересно`, если не хочешь видеть её снова
- если карточка уже имеет статус отклика, можно нажать `Скрыть`
- при включённом автоскрытии такие карточки будут убираться автоматически

### 2. Скрывать вакансии и компании на странице вакансии

- на самой странице вакансии используй `Скрыть вакансию` или `Скрыть компанию`
- если объект уже скрыт, расширение покажет это отдельным статусом

### 3. Хранить заметки

- открой popup расширения
- нажми `Открыть заметки`
- создай заметку с названием и текстом
- заметки можно редактировать и удалять

### 4. Подставлять заметки в форму отклика

- в поле сопроводительного письма нажми `Выбрать из моих заметок`
- выбери нужную заметку из списка
- если нужно, открой полный экран заметок через `Посмотреть все`

### 5. Сохранять текст из формы отклика

- под нужным `textarea` включи `Сохранить на потом`
- задай название заметки
- нажми `Откликнуться`
- при отправке формы расширение сохранит только те поля, где была отмечена галка

### 6. Переписывать текст через Gemini

- открой popup расширения
- добавь Gemini API key из Google AI Studio
- выбери модель Gemini 3+ и режим переписывания
- вернись к форме отклика на hh.ru
- нажми `Переписать с Gemini` рядом с нужным `textarea`
- расширение отправит текст в Gemini только по этому нажатию и заменит содержимое поля готовым вариантом

## Безопасность и данные

- расширение не отправляет ваши заметки, токены, cookies, резюме, отклики и другие персональные данные на сторонние серверы
- заметки хранятся локально в storage браузера
- запросы из расширения отправляются только на сам `hh.ru`, чтобы скрывать вакансии и получать их текущий статус скрытия
- Gemini API key хранится локально в storage браузера
- текст из `textarea` отправляется в Gemini API только по явному нажатию кнопки `Переписать с Gemini`
- расширение не содержит собственного удалённого backend, аналитики или облачного хранилища

## Технические детали

- скрытие вакансий и компаний работает через внутренние blacklist API hh.ru
- состояние скрытия на странице вакансии тоже проверяется через hh.ru API
- список вакансий и формы отклика отслеживаются на динамически изменяющейся странице через `MutationObserver`
- popup открывает отдельную страницу заметок через стандартные возможности браузерных расширений
- хранение заметок реализовано через совместимый extension storage API браузера
- Gemini rewrite вызывается через background service worker расширения и Gemini API
- системный prompt и список поддерживаемых Gemini-моделей лежат в отдельном файле `prompts/gemini-rewrite.json`
- на тегах `v*` GitHub Actions автоматически собирает отдельный Chromium ZIP и отдельный Firefox ZIP
- шаблон changelog для следующих релизов лежит в `CHANGELOG_TEMPLATE.md`

## Ограничения

- Safari сейчас не поддерживается
- Firefox пока распространяется только как временно загружаемое дополнение, а не через магазин
- разметка hh.ru может меняться, поэтому часть логики сделана с fallback-селекторами
- если hh.ru сильно изменит разметку формы отклика, селекторы для inline-заметок могут потребовать обновления
